# Projet HRNET

## Pré-requis

Node JS : https://nodejs.org/en

## Installation 

1. Cloner le repository:

git clone https://github.com/

2. Installer toutes les dépendances:

###### npm install ou yarn

3. Lancer le front-end:

###### npm start ou yarn start

## Dépendances utilisés

React
React-Router
React-Table
