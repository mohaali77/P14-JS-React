# Projet HRNET

## Pré-requis

Node JS : https://nodejs.org/en

## Installation 

1. Cloner le repository:

git clone https://github.com/mohaali77/P14-JS-React

2. Installer toutes les dépendances:

#### npm install ou yarn

3. Lancer le front-end:

#### npm start ou yarn start

React

React-Table
